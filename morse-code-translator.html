关于在线摩斯密码双向翻译器

morse-code-translator 是一款面向学习、通信练习与内容编码场景的在线摩斯密码翻译器。它可以将常见英文文本、数字和标点快速转换为由点和划组成的摩斯电码，也能把规范输入的摩斯密码还原为可读英文，解决手动查询编码表效率低、容易出错的问题。对于正在学习 Morse Code 的用户，这个工具不仅提供英文转摩斯密码、摩斯密码转英文的双向转换，还内置基于 Web Audio API 的摩斯密码提示音播放功能，便于通过听觉记忆点划节奏。使用方法非常简单：在输入区粘贴英文句子或摩斯代码，选择自动识别或指定转换方向，点击“翻译”即可获得结果；如需练习听写，可点击“播放”并调节点时长控制速度。本工具采用响应式设计，手机和桌面浏览器均可直接使用，适合作为在线摩斯电码转换工具、摩斯密码学习辅助工具以及轻量级 Morse Code Translator。
