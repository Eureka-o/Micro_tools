关于 text-to-speech-test 浏览器原生 TTS 测试工具

text-to-speech-test 是一款面向开发者、内容创作者和产品测试人员的在线文字转语音工具。它直接调用浏览器内置的 window.speechSynthesis API，无需安装插件，也不需要上传文本到服务器，即可完成浏览器原生 TTS 测试。对于需要快速验证 Web 语音合成效果、检查不同语音角色、测试中文朗读清晰度以及调整朗读语速和音调的场景，本工具能够显著降低调试成本。

使用方法非常简单：在输入框中粘贴或输入需要朗读的文本，从语音角色列表中选择浏览器提供的声音，然后拖动语速、音调和音量滑块，最后点击“开始朗读”。如果朗读过程中需要中断、暂停或继续，也可以通过对应按钮实时控制。由于不同系统和浏览器内置的语音库不同，实际可选声音会有所差异。无论是进行在线文本转语音测试、Web TTS 功能兼容性验证，还是制作语音交互原型，text-to-speech-test 都能提供轻量、直观、隐私友好的测试体验。

基于浏览器原生 SpeechSynthesis API 构建，所有文本仅在本地浏览器中处理。
