实用指南

本地 LaTeX 公式渲染器

用本地 KaTeX 文件预览公式，离线检查数学笔记、论文片段和教学示例。

使用方法

  1. 在编辑器中输入或粘贴 LaTeX 公式。
  2. 使用内置本地 KaTeX 运行库和字体预览渲染结果。
  3. 调整 display mode、矩阵语法或复杂表达式，直到输出符合文档需要。
  4. 复制或导出公式，用于笔记、Markdown、幻灯片或说明文档。

常见场景

  • 把公式写进 Markdown、LaTeX 或静态站点前先检查显示效果。
  • 在本地渲染矩阵、求和、积分和对齐公式。
  • 离线准备透明背景公式图片。

本地运行说明

公式在浏览器内渲染，草稿、课堂笔记和论文片段不会上传。

常见问题

它依赖 KaTeX CDN 吗？

不依赖。JavaScript、CSS 和字体文件都存放在本地 assets/vendor 目录。

为什么离线还需要字体文件？

KaTeX 依赖专用字体保证字形和间距正确，字体已本地打包。

能编译完整 LaTeX 文档吗？

不能。它用于公式渲染，不是完整 TeX 工程编译器。
