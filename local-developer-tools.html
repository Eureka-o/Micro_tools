隐私优先本地工具箱。

一个好用的开发者工具箱，应该能在三十秒的小任务里快速打开，也应该足够谨慎，不把你粘贴的文本泄露出去。这个合集收纳了 Micro Tools Matrix 中最通用的页面，覆盖编码、调试、写作、文档和技术学习。

你可以用正则工具检查模式，用文本对比做 review，用 ASCII 树生成 README 结构，用 LaTeX 渲染器处理数学笔记，用 JSON 转换工具清理数据，用 JWT 解码器调试鉴权，用 SEO 标签审计器检查静态页面。每个工具都刻意保持小而明确：完成单一任务、输出可复制结果，不把快速检查变成沉重流程。

本站定位统一为隐私优先本地工具箱。这意味着本地脚本、默认不上传、离线友好资源，以及为未来赞助预留干净结构但不加重当前体验。

这个合集也是不确定需求时的通用入口：技术写作、调试、数据清理、安全检查和静态站 SEO 都能从这里继续进入。它让站点更容易被记住，也鼓励用户在每天出现的小维护任务中反复回来。
