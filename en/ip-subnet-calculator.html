This IP/CIDR subnet calculator is built for network engineers, DevOps teams, cloud administrators, and developers. After you enter an IPv4 address and prefix length, the page uses plain browser JavaScript to run binary network math and show the network address, broadcast address, subnet mask, usable IP range, usable host count, total addresses, wildcard mask, and binary network address. All input remains in the current page and is never uploaded to a server.
