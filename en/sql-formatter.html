SQL 格式化工具是 Micro Tools Matrix 提供的免费在线开发者工具，适合在排查查询语句、整理数据库脚本、编写文档或代码评审前快速统一 SQL 风格。你可以把压缩成一行或缩进混乱的 SELECT、JOIN、WHERE、GROUP BY、ORDER BY 等语句粘贴进来，页面会在浏览器本地处理文本，输出更易阅读的层级排版，并对常见关键字进行高亮。整个过程无需上传、无需登录，也不会把 SQL 内容发送到服务器，适合处理包含表名、字段名或业务条件的敏感片段。工具支持复制结果、下载 .sql 文件、示例填充和清空重置，帮助你用轻量方式完成 SQL 美化与格式化。
