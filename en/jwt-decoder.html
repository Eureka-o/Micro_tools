JWT 本地解码器是 Micro Tools Matrix 为开发者准备的免费在线开发者工具，用于快速查看 JSON Web Token 的 Header 与 Payload。你可以把登录态调试、接口联调或权限排查中的 JWT 粘贴到页面中，工具会在浏览器本地处理并格式化 JSON，帮助你确认 alg、typ、iss、sub、aud、exp、iat 等字段。整个过程无需上传、无需注册，也不会把令牌内容发送到服务器，适合处理敏感测试数据。请注意，本工具只负责解码展示，不验证签名真伪，也不判断令牌是否可信；在生产环境排查安全问题时，仍应结合后端密钥、证书与授权策略进行校验。
