JSON 工具合集

隐私优先本地工具箱。

JSON 处理通常不大，却经常很敏感：API 响应、Webhook payload、配置文件、测试样本和课程数据里可能包含 token、用户标识或内部字段。这个合集把常见 JSON 工作留在浏览器里完成，覆盖格式化、压缩、路径提取、JSON 转 CSV、YAML 互转和表格清理等场景，不要求把内容上传给远程解析器。

它不是为了取代 IDE 或完整数据平台，而是优化日常中间步骤：粘贴响应、校验结构、展开字段、复制干净结果，然后继续调试。开发者可以在联调时检查 API 样本，学生可以理解嵌套数据结构，工程师也可以在断网环境里整理小型配置片段。

Micro Tools Matrix 的定位是隐私优先本地工具箱。运行库统一本地化，重型脚本按需加载，导出文件通过浏览器 Blob 完成。需要快速转换但不想上传 JSON 时，这个合集就是入口。

对搜索用户来说，这个页面也是一张路线图：你不必停在某个孤立转换器里猜还有哪些工具可用，而是可以从校验进入字段提取，再继续转成 CSV 或 Markdown 表格。真实数据清理往往需要连续两三步，把这些入口放在一起，可以减少来回搜索，也更容易确认结果。

常见问题

这些 JSON 工具会上传数据吗？

不会。解析和转换都设计为在浏览器内存中完成。

我应该从哪个工具开始？

校验和清理用 JSON 格式化/压缩，提取字段用 JSON Path，转表格用 JSON 转 CSV。

可以离线使用吗？

站点静态资源和本地供应商库可用后，页面按离线友好方式运行。
