关于我的 IP 地址查看器

ip-address-lookup 是一个免费、轻量、无需安装的在线公网 IP 地址查看器，适合需要快速确认网络出口地址的个人用户、站长、运维工程师与跨境办公人员。很多时候，本机显示的内网地址并不能代表互联网上看到的真实地址，而本工具会通过纯前端调用公开 IP API，帮助你立即查看我的公网 IP 地址，并展示大致国家地区、城市、运营商、时区和经纬度信息。使用方法非常简单：打开页面后点击“检测我的公网 IP”，即可自动完成查询；如果你需要进行指定 IPv4 或 IPv6 地址地理位置查询，也可以在输入框中填入 IP 后点击查询。该在线 IP 查询工具无需注册账号，不会依赖后端服务器，适合排查代理、VPN、CDN、远程办公网络异常以及验证网站访问来源等场景。
