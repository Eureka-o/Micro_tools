关于 jwt-decoder 本地 JWT 解码工具

jwt-decoder 是一款面向前端开发者、后端工程师与接口测试人员的在线 JWT 本地解码器，用于快速解析 JSON Web Token 的 Header 与 Payload。很多调试场景需要查看 alg、typ、iss、sub、aud、exp 等声明，但又担心把敏感 Token 上传到第三方服务。本工具的核心优势是纯前端运行，所有 JWT 解析都在浏览器内完成，不向服务器发送任何数据，适合登录鉴权排查、API 联调、OAuth 调试和过期时间检查。使用时只需将完整 JWT 粘贴到输入框，点击“解码”即可获得格式化 JSON 与常用声明说明。需要注意的是，JWT 解码不等同于签名验证，本工具帮助你安全查看令牌内容，但不能证明 Token 未被篡改。
